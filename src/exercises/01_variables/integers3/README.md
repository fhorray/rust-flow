# Numeric Literals

## Description

Rust supports different ways to write numbers:

- Decimal: `98_222`

- Hex: `0xff`

- Octal: `0o77`

- Binary: `0b1111_0000`

- Byte (u8 only): `b'A'`

Your task is to define variable `x` as `100` using a decimal literal,

and variable `y` as `255` using a hex literal.

## Hints

1. Hex literals start with `0x`.
