# Select All Users

Your goal is to retrieve all records from the `users` table.

## Schema
The table `users` has the following columns:
- `id` (integer)
- `name` (varchar)
- `role` (varchar)

## Task
Write a SQL query in `query.sql` to select all columns (`*`) from the `users` table.

## Expected Output
You should see 3 users: Alice, Bob, and Charlie.
