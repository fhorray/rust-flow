# Features Demo Course

This course showcases all the powerful features of the Progy platform.

## Features:
- **Module Icons**: Beautiful Lucide icons in the sidebar.
- **Custom Branding**: Cover images and primary colors.
- **Exercise Metadata**: Difficulty dots and custom tags.
- **Achievements**: Unlockable rewards for your progress.
- **Personalized Messages**: Completion messages that know your name.
- **Dynamic Layouts**: Choose between Grid and Vertical layouts.
